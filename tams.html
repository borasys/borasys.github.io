TAMS 소개

TAMS(Telephony Assurance by Message Scripts)는 SIP Message를 사용하여 단말과 서버간의 메세지를 자동적으로 실행하는 스크립터이다. SIP 프로토콜의 Request Message와 Response Message를 구분하여 Scenario를 작성하여 단말의 서비스 상태 유무를 확인할 수 있다.

TAMS 특징

-Simple SIP message tester -Editable Timer -Change response message -Select response code -Generator media (711,723, 729) -Interworking TAS & PE 11 -Register, authorization test -Quick start quide -UAS / UAC emulation -Signaling Aging test -RFC3261, 3262, 3264, 3265, 3311, 3325, 3515, 3842, 3881, 4028, 2617

TAMS 기대효과

-VoIP 단말의 안정적 검증을 가능하게 함 -프로토콜의 예외사항에 대한 대응 능력 향상 -새로운 부가가치 서비스의 안전적 운영을 통한 이익 창출 -제공하는 검증된 자료를 가지고 관리함으로써 기술적인 체계정립과 우수성을 제시함 -종단 측정, 단말 등을 활용함으로써 복잡하고 다양한 프로토콜 특성 파악가능 -유일하고 차별화된 결과치를 제시 -TAS연동에 따른 음성품질과 프로토콜 검증을 단일 시스템에서 측정 -PE-11연동에 의해서 동시에 복수 단말에 대한 품질측정(MOS) 가능
